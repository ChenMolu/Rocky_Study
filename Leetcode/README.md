树节点：
```java
class TreeNode {
    int value;
    TreeNode left;
    TreeNode right;
    public TreeNode () {};
    public TreeNode(int value){
        this.value = value;
        this.left = null;
        this.right = null;
    }
}
```

链表节点：
```java
class Node {
    int value;
    Node next;
    public Node() {};
    public Node(int value){
        this.value = value;
        this.next = null;
    }
}
```
